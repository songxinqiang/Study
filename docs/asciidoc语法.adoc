= 有关asciidoc的语法

== 标题

标题有两种标记方式，两行式和一行式，都支持最多四级标题，分别是

 书名
 =====
 作者名
 v0.1, 2012-12
 :doctype: book

 第二章标题
 ----------
 内容……

 第二章二级标题
 ~~~~~~~~~~~~~~
 内容……

 第二章三级标题
 ^^^^^^^^^^^^^^
 内容……

 第二章四级标题
 ++++++++++++++
 内容……

 = 书书名 =
 == 第一章标题 ==
 === 第一章二级标题 ===
 ==== 第一章三级标题 ====
 ===== 第一章四级标题 ======

注意第二种单行的风格右边的那部分标记也可以省略的
